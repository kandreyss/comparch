\chapter{ТЕКСТ ЗАДАНИЯ}

Практическая часть работы включает выполнение следующих действий:

\begin{enumerate}

    \item формирование числовых значений в соответствии с индивидуальным заданием, 
    перевод их в шестнадцатеричную систему счисления и определение минимального
    формата представления исходных данных как целых чисел;

    \item запись целочисленных данных в РОН;
    
    \item по заданному алгоритму составление и выполнение простейшей программы с
    целочисленными данными, хранящимися в РОН;

    \item выполнение программы;
    
    \item контроль результатов работы программы.

\end{enumerate}

Правильность разработки и выполнения программ арифметико-логической обработки
данных контролируется путем ручной трассировки заданных алгоритмов с последующим
сравнением результатов работы программ с результатами ручной трассировки.
